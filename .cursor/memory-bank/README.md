# Memory Bank - NaturaPets

Ce dossier centralise la memoire projet pour guider l'agent de facon coherente entre les sessions.

## Objectif

- Eviter de re-expliquer les conventions a chaque demande.
- Garder des decisions stables sur le theme WordPress.
- Accelerer les implementations et revues.

## Fichiers

- `project-context.md` : stack, architecture et contraintes globales.
- `coding-standards.md` : standards techniques a respecter.
- `workflow-checklists.md` : checklists operationnelles avant livraison.
- `decision-log.md` : journal des choix importants.

## Regles d'utilisation

1. Lire `project-context.md` avant toute modification importante.
2. Appliquer `coding-standards.md` par defaut.
3. Utiliser `workflow-checklists.md` en fin de tache.
4. Ajouter une entree dans `decision-log.md` quand une decision durable est prise.
